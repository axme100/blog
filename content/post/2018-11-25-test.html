Introduction

In this post, I’m going to show off my first full-stack, data viz web application that allows you to visualize usage data of the Totago, an online program for helping people to access hiking trails via public transportation. My application is an interactive, choropleth map where each map area is colored according to the number of users who generated Totago iteneraries from that area. When users click on thee map areas (postal codes or neighborhoods), information about the destinations originaing from the areas is displayed dynamically through resizing cirlces on the map as well as an adjacent bar graph.

[Here is a gif showing the page in action]

You can find the live version of the application here (hosted on Heroku).

Also, the code is located in this GitHub repository

The application, hosted on Heroku, uses Python and Flask for the backend code and connects to a Postgres database hosted on AWS RDS with SQL Alchemy. The front retrieves data via AJAX calls which is visualized by a in the map element (powered by Map Box GL JS) and a a bar graph (Mapbox GL JS). You can find all the source code in the link above.
